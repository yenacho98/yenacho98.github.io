Automated Classification of Vegan Products Using Ingredient Data Data Project

Ingredient lists for packaged foods are often lengthy and difficult to interpret, making it challenging for consumers—especially those with dietary restrictions—to quickly identify suitable products. While public data exists, it is not structured for easy dietary filtering.

This project uses publicly available data from the USDA FoodData Central (Branded Foods) database to develop a scalable approach for identifying likely vegan products through automated ingredient text analysis.

Methodology:

  • Scraped ~1,000 products (40 pages) using Easy Scraper
  • Extracted product name, brand, category, and ingredient lists
  • Cleaned and processed unstructured ingredient text
  • Computed ingredient-level features (e.g., ingredient count)
  • Built a rule-based classifier to flag likely vegan products based on absence of animal-derived ingredients

This project demonstrates how unstructured consumer product data can be transformed into actionable insights for dietary decision-making.

Morning Reattachment and Daily Work–Life Balance

Developing and Testing a Process-Focused Model to Explain How Morning Reattachment Facilitates Daily Work–Life Balance Satisfaction

This project develops and tests a process model explaining how morning reattachment—mentally reconnecting to work at the start of the day—shapes daily work–life balance satisfaction. Drawing on the episodic process model of task performance and the action regulation model of work–life balance, this research examines how early-day cognitive processes influence downstream outcomes.

Social Media Breaks and Work Focus

My current thesis examines how social media use during the workday functions as a microbreak, and whether it facilitates or hinders recovery, attentional control, and work focus using daily diary and multilevel modeling approaches.

Daily Stress and Work–Life Balance

This work investigates how daily stressors and strains influence work–life balance perceptions, using multilevel analyses to capture within-person fluctuations across time.

Commuting and Recovery

This line of research explores how commuting experiences shape recovery, affective rumination, and subsequent workday performance, integrating physiological and survey data.
